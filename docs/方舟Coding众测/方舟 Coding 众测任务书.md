常见问题

* 任务频繁重启、运行一段时间后又中止，怎么办？
  
  * 请将任务 ID 提供给 @王竞恺 进行排查

* 如果我上传了 5 个任务，只对其中 3 个做评分，可以吗？
  
  * 评分反馈三个任务就可以

* 只在综合打分下面写评论，不在产出评价的轨迹分析下额外评论和备注行吗？
  
  * 为了帮助我们更好地理解你对模型细节产出的评估逻辑，建议结合产出针对性提供反馈，尤其是对模型有明显亮点和错误的地方备注评论；更加专业和细节的备注是我们后期质检的加分项

* 自动注册的 Xpert 账号，提示昵称已被使用，为什么绑定手机号 / 实名认证时会有问题？
  
  * 自动注册的账号登录后，只能修改密码和退出登录，不能直接在那个账号上完成手机号绑定和实名认证
  
  * 如果通过手机验证码登录时提示“该昵称已被使用”，这件事本身不影响继续操作
  
  * 但后续如果是重新取一个昵称进入平台完成实名认证，不能关联回之前提交任务的那个账号

* 任务重复上传了，或者想换题重新测，能不能删除？
  
  * 可以不用删，放着就行

* 任务上传/下载失败，怎么办？
  
  * 首先请尝试通过文件夹的方式上传任务，通常更快
  
  * 如果上传下载反复失败，请联系 @王竞恺 协助
    
    

质检说明
====

**什么样的任务打分，不能通过质检**
-------------------

* **打分没有区分度：**如果测试过程中给所有模型打分都趋同，没有区分出好模型和可能有问题的模型，会被质检重点检查反馈质量

* **打分与产出实际表现不一致**：产出质量明显很差，打分却很高，打分完全不符合事实情况

* **模型打分偏序与大盘显著不同：**我们会对开发者大盘打分进行统计，如果开发者所提供的打分与大盘有显著差异，会被质检重点检查反馈质量

* **用 AI 生成反馈：**开发者测试活动的目的是获得真实的场景和专家反馈，如果开发者完全依赖 AI 检验模型产物并生成评价，而没有自己测试产物，会被质检检查反馈质量，被认为不合格
